# Space Geometry Dash

宇宙を舞台にしたジオメトリダッシュ風の2Dアクションゲームです。

## 特徴

- **多彩な障害物**: スパイク、浮遊障害物、レーザー、回転刃、壁など15種類以上
- **特殊ギミック**:
  - ジャンプ台（スーパージャンプ）
  - ジャンプ雲（空中ジャンプリチャージ）
  - 偽物の壁（真ん中だけ乗れるトラップ）
  - 一マスの壁（小さな足場）
- **高難易度チャレンジ**:
  - 階段チャレンジ（階段＋ジャンプ雲＋ゴール足場）
  - ジャンプ雲チャレンジ（空中飛び移り）
  - ジャンプ台チャレンジ（スーパージャンプで高所へ）
- **パワーアップ**:
  - スピードブースト（加速）
  - スローモーション（減速）
  - 無敵アイテム
- **動的な地形**: 波打つ地形が自動生成
- **パーティクルエフェクト**: 爆発、トレイルなど

## 操作方法

- **スペースキー / クリック**: ジャンプ（最大2回まで空中ジャンプ可能）
- **ジャンプ台**: 上から踏むとスーパージャンプ
- **ジャンプ雲**: 触れた状態でジャンプするとジャンプ回数リセット

## 遊び方

1. `index.html` をブラウザで開く
2. 画面をクリックしてゲーム開始
3. 障害物を避けながら進む
4. できるだけ長く生き延びてハイスコアを目指す

## 技術スタック

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## ファイル構成

```
pj/
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css      # スタイルシート
├── js/
│   ├── game.js        # ゲームメインループ
│   ├── player.js      # プレイヤークラス
│   ├── obstacle.js    # 障害物とマネージャー
│   ├── terrain.js     # 地形生成
│   ├── particle.js    # パーティクルシステム
│   ├── powerup.js     # パワーアップシステム
│   └── main.js        # エントリーポイント
└── README.md          # このファイル
```

## 開発

ローカルサーバーで実行する場合：

```bash
# Node.jsがインストールされている場合
node server.js
# ブラウザで http://localhost:3000 にアクセス
```

または、単純に `index.html` をブラウザで直接開くこともできます。
